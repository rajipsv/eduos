# Pillar 2 — Academy OS

**Product:** EduOS · **Layer:** 2 · **Sidebar:** Academy

---

## Purpose

Academy OS runs the **core teaching operations** of your center: tutors, batches, auto-generated schedules, students, fees, attendance, and tests. Live classes happen on Zoom/Meet/Teams — EduOS coordinates **when**, **who**, and **what's next**.

---

## Who uses it

| Role | Access |
|------|--------|
| Center admin | All Academy modules |
| Teacher | Own batches, schedule, attendance, students |
| Student / Parent | Read-only schedule (via their portals) |

---

## Modules & functionalities

| Module | Path | Key features |
|--------|------|--------------|
| Batches | Academy → Batches | Create batches, assign teacher, pick availability slot, topics → auto-schedule |
| Class Schedule | Academy → Class Schedule | Week calendar, cancel/reschedule, mark done, meeting links |
| Teachers | Academy → Teachers | Tutor roster, subjects, batch load, reports |
| Students | Academy → Students | Records, batch assignment, parent contacts, fee overrides |
| Fees & Invoices | Academy → Fees & Invoices | Monthly billing, generate invoices, reminders, mark paid |
| Attendance | Academy → Attendance | Present / absent / late, optional parent WhatsApp |
| Tests & Marks | Academy → Tests & Marks | Record scores, averages, share with parents |

## How to use — step by step

### 1. Teachers

1. **Academy → Teachers → + Add Teacher** — name, email, phone, subjects.
2. If you (owner) also teach: use **Settings → My teaching profile** instead of a separate login.
3. Set availability in **Tutor Success → Schedule → Manage slots** (required for slot-based batch scheduling).

### 2. Create a batch

1. **Academy → Batches → + New Batch**.
2. Set name, **Assigned teacher**, capacity, monthly fee, fee due day, subjects.
3. Under **Class schedule from teacher availability**, pick a **free slot** (or manual days/times if no slots).
4. Set course start date and meeting platform (Google Meet / Zoom / Teams).
5. Enter **topics** — one per line (one class per topic).
6. Click **⚡ Auto-Generate Schedule**, review preview, **Save Batch**.

**Rules:** Each teacher availability slot can only be assigned to one batch. Booked slots are hidden when creating another batch.

### 3. Add students

1. **Academy → Students → + Add Student**.
2. Assign batch; enter parent name and phone (used for fees and comms).
3. Optional: override monthly fee on student record.

### 4. Daily schedule management

1. **Academy → Class Schedule** — Calendar or List view.
2. Filter by batch; click a session.
3. Actions: **Join Meeting**, **Copy Link**, **Reschedule**, **Cancel**, **Mark done**.

### 5. Mark attendance

1. **Academy → Attendance** — select batch and date.
2. Toggle each student: Present / Absent / Late.
3. Check **Notify parents via WhatsApp** if desired → **Save Attendance**.

### 6. Record tests

1. **Academy → Tests & Marks → + Record Test**.
2. Enter test name, batch, subject, date, max marks.
3. Enter each student's score → save.
4. Results feed reports and parent notifications.

### 7. Fees & invoices

1. Configure defaults in **Settings → Billing & Invoices** (prefix, UPI, due day).
2. Set batch **Monthly fee** on each batch.
3. **Academy → Fees & Invoices → Generate invoices** for the month.
4. **Send** reminders; **Mark paid** or **Confirm payment** when parent reports UPI transfer.
