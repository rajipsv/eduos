Pillar 4 — Student Success

Product: EduOS · Layer: 4 · Sidebar: Student Success

Purpose

Student Success tracks learning outcomes beyond attendance: success scores, skill journeys, interventions, homework, certificates, and parent progress summaries. It answers: *Is this student on track, and what should we do if not?*

Who uses it

RoleAccess
Center adminFull engine — all tabs and admin actions
TeacherAssignments, feedback, certificates (via Tutor Success overlap)
ParentHomework view only (read-only list)
StudentOwn homework — view and submit

Modules & functionalities

Path: Student Success (admin/teacher view)

Admin toolbar actions:

  • + Assignment — create homework for a batch/student
  • Issue Certificate — award completion or achievement certificate
  • + Feedback — teacher notes visible to parents

Tabs:

TabFunction
OverviewAll students — success score, attendance, tests, homework %, risk badge
JourneySkills by subject, progress bars, projects, certificates
InterventionsAt-risk plans — generate, apply, mark resolved
AssignmentsHomework list — grade submissions, track due dates
Parent SummaryDraft progress summary → Send to Parents

Success score combines attendance, test averages, curriculum progress, and homework completion.

Risk levels: High (&lt;60 score), Medium, Low — drives intervention suggestions.

How to use — step by step

A. Review all students

  1. Open Student Success → Overview.
  2. Scan the table for red high risk badges.
  3. Click a student row to open their Success Profile below.

B. Track learning journey

  1. Select student from dropdown (top toolbar).
  2. Journey tab — review skills table (subject, level, progress %).
  3. Add projects or view certificates earned.

C. Apply an intervention

  1. Interventions tab for an at-risk student.
  2. Review suggested plan (extra practice, parent call, etc.).
  3. Click Generate & Apply Plan.
  4. When resolved, click Mark resolved.

D. Manage homework

  1. + Assignment — title, batch, due date, description.
  2. Assignments tab — see status per student (pending, submitted, graded).
  3. Click Grade to enter marks and feedback.
  4. Students submit via Student Success (student login); parents can mark submitted (parent confirmation).

E. Send parent summary

  1. Parent Summary tab for selected student.
  2. Review auto-generated summary text.
  3. Click Send to Parents (uses Communication engine).

F. Issue certificate or feedback

  1. Issue Certificate — pick student, title, date.
  2. + Feedback — short note for parent portal Feedback tab.

Verify

  • [ ] Overview shows all enrolled students with scores
  • [ ] New assignment appears in Assignments tab and student/parent homework views
  • [ ] Graded homework updates homework % on overview
  • [ ] Intervention can be applied and resolved
  • [ ] Parent summary sends to message log
