# Pillar 5 — Tutor Success

**Product:** EduOS · **Layer:** 5 · **Sidebar:** Tutor Success

---

## Purpose

Tutor Success is the **teacher workspace**: lesson plans, homework review, schedule, availability slots, performance metrics, and professional development. It connects teachers to batches without replacing Zoom/Meet for live teaching.

---

## Who uses it

| Role | Access |
|------|--------|
| Center admin | All tutors (overview table) + own profile if linked as teacher |
| Teacher | Own batches, schedule, homework queue, lesson plans |

---

## Modules & functionalities

**Path:** Tutor Success

**Teacher selector** (admin): pick any tutor from dropdown. Defaults to your linked profile if **Settings → I also teach classes** is enabled.

**Tabs:**

| Tab | Function |
|-----|----------|
| Overview | Tutor roster table — batches, students, score, pending review, upcoming class |
| Performance | Batch progress, completion rates, student counts |
| Lesson Plans | Create, AI-generate, deliver, delete lesson plans |
| Homework | Submissions awaiting grade — **Grade** button per student |
| Schedule | Upcoming classes + **Manage slots** for availability |
| Development | Log PD activities (hours, type, status) |

**Availability slots** (critical for Academy batch scheduling):

- Each slot = days + start/end time
- Multiple slots per teacher (e.g. weekday evening + Saturday morning)
- **Manage slots → Save availability**
- Booked slots hidden when admin creates another batch for same teacher

---

## How to use — step by step

### A. Set your availability (teacher or owner-teacher)

1. **Tutor Success → Schedule tab**.
2. Click **Manage slots**.
3. Add slot: select days (Mon/Wed/Fri), start time, end time.
4. Add more slots if needed → **Save availability**.
5. Confirm slots display in Availability panel.

### B. Review today's work (teacher login)

1. Log in as **Teacher** (`anita@tutorhub.com` / `demo123`).
2. **Teacher Today** shows next classes and tasks.
3. **My schedule** mirrors Academy schedule for your batches only.

### C. Create a lesson plan

1. **Tutor Success → Lesson Plans tab**.
2. **+ Lesson Plan** — manual entry, or **AI Lesson Plan** for draft.
3. Fill topic, objectives, activities, homework.
4. After class: **Mark delivered**.

### D. Grade homework

1. **Homework tab** — table of pending submissions.
2. Click **Grade** — enter score and comments.
3. Graded work updates Student Success and parent views.

### E. Track performance (admin)

1. **Overview tab** — compare tutors on batches, students, pending reviews.
2. **Performance tab** — batch-level curriculum progress bars.

### F. Log professional development

1. **Development tab → + Log PD Activity**.
2. Enter title, type, hours, date, status.

---

## Owner who also teaches

1. **Settings → My teaching profile** — enable **I also teach classes**.
2. Tutor Success opens on your profile automatically.
3. Set availability here before creating batches assigned to yourself.
